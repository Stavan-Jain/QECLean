## 1. Statement

**Theorem A13.** *Let `G̃` be any finite abelian group, `σ ∈ G̃` of order 2,
and `A, B ∈ R̃ = F₂[G̃]` arbitrary. Then:*

1. *(Composite vanishing)* `δ₁∘δ₂ = 0`.
2. *(Bockstein equality)* `dim_F₂ (1+σ)·H₁(cover) = k̃ − k` exactly.
3. *(Deck-module structure)* `H₁(cover) ≅ D^{k̃−k} ⊕ F₂^{2k−k̃}` as a
   `D`-module. Together with Theorem A12 ((R) ⟺ `k̃ = k` ⟺
   `1+σ ∈ (A,B)`), this is the complete deck-module classification of the
   logical space of a free-ℤ₂ BB cover from the two ranks `k, k̃` — and it
   covers twisted decks (`σ = x^a y^b`) and arbitrary coefficient blocks
   `S[P]`, not just axis doublings.
4. *(Element form)* Whenever `Az, Bz ∈ εR̃`, every valid
   `W = A·ε⁻¹(Bz) + B·ε⁻¹(Az)` lies in `ε(A,B)`; moreover there is a
   canonical choice (any ℤ/4-compatible one) with `W = 0` on the nose.

The bridge between 1 and 2 is the **defect identity** (unconditional, from
the A12 LES bookkeeping `im p₁ = ker δ₁`, `ker τ₁ = im δ₂`,
`ε_* = τ₁∘p₁`):

```
E − (k̃ − k) = dim δ₁(im δ₂) = rank(δ₁∘δ₂),
```

since `E = dim ker δ₁ − dim(ker δ₁ ∩ im δ₂)` and
`k̃ − k = dim ker δ₁ − dim im δ₂`. Claim 3 follows because every finite
`D`-module is `D^a ⊕ F₂^b` with `a = dim εM`, `2a + b = dim M`.

## 2. Proof of the composite vanishing (the tower lift)

**Frattini lift.** Decompose `G̃ = ∏_t Z_{n_t}`; `σ = (σ_t)` with each
`σ_t ∈ {0, n_t/2}`. Pick any `t` with `σ_t ≠ 0` and set
`Ĝ = ⋯ × Z_{2n_t} × ⋯` (double that factor, same exponent tuple for
`σ̂`). Then `σ̂ ↦ σ` under `Ĝ ↠ G̃ = Ĝ/⟨σ̂²⟩` and `σ̂` has order exactly 4
(`2σ̂ = (0,…,n_t,…,0) ≠ 0`, `4σ̂ = 0`). For the standard free-ℤ₂ BB cover
this is literally the next rung of the doubling tower:
`Ĝ = Z_{4ℓ}×Z_m`, `σ̂ = x^ℓ`. Write `R̂ = F₂[Ĝ]`, `ε̂ = 1+σ̂`, and lift
`A, B` to `Â, B̂ ∈ R̂` (any lifts; the canonical same-exponent lifts will
do). Three facts, all consequences of `R̂` being **free as a module over
`Λ = F₂[⟨σ̂⟩] ≅ F₂[ε̂]/(ε̂⁴)`** (group algebra over a subgroup algebra,
coset basis):

- **F1.** `ε̂² = 1+σ̂²`, so `R̂/ε̂²R̂ ≅ F₂[Ĝ/⟨σ̂²⟩] = R̃`, sending
  `ε̂ ↦ ε` and `Â ↦ A`, `B̂ ↦ B`.
- **F2.** `ker(R̂ → R) = ε̂R̂`, where `R̂ → R = F₂[Ĝ/⟨σ̂⟩]`: the kernel is
  the ideal generated by `{1+σ̂^j}`, and `1+σ̂² = ε̂²`,
  `1+σ̂³ = ε̂ + σ̂ε̂²` both lie in `(ε̂)`.
- **F3.** `Ann_{R̂}(ε̂) = ε̂³R̂` (in `Λ`: `(1+σ̂)w = 0` forces
  `w ∈ (1+σ̂+σ̂²+σ̂³) = (ε̂³)`; freeness transfers this to `R̂`).
  Likewise `Ann_{R̃}(ε) = εR̃` over `D`.

**The computation.** Let `z̄ ∈ H₂(K̄)`, i.e. `Āz̄ = B̄z̄ = 0` in `R`. Lift
`z̄` to `ẑ ∈ R̂`. By F2, `Âẑ = ε̂â` and `B̂ẑ = ε̂b̂` for some
`â, b̂ ∈ R̂`. Reduce mod `ε̂²` (F1): `z := ẑ mod`, `a := â mod`,
`b := b̂ mod` satisfy `εa = Az`, `εb = Bz` in `R̃` — a valid chain of
choices for the two connecting-map recipes: `δ₂(z̄) = [(b̄, ā)]`, and
`δ₁` of that class may be computed with the lift `(b, a)`, whose boundary
is `∂₁(b,a) = A·b + B·a =: W`. Now lift `W` to `Ŵ := Â·b̂ + B̂·â`:

```
ε̂·Ŵ = Â(ε̂b̂) + B̂(ε̂â) = Â·B̂ẑ + B̂·Âẑ = 2·ÂB̂ẑ = 0     (char 2),
```

so `Ŵ ∈ Ann_{R̂}(ε̂) = ε̂³R̂ ⊆ ε̂²R̂` (F3), i.e. **`W = 0` in `R̃`**.
The connecting-map output is the preimage of `W` under the isomorphism
`R ≅ εR̃` (mult-by-`ε`; injective by F3-for-`R̃`), so
`δ₁δ₂(z̄) = [0] = 0`. Since each `δ` is well-defined on homology,
computing the composite along one valid chain of choices suffices. ∎(1)

Claim 4's "every valid `W`": the ambiguities are `a ↦ a + t`,
`b ↦ b + t'` with `t, t' ∈ Ann_{R̃}(ε) = εR̃` (moving `W` by
`B·t + A·t' ∈ ε(A,B)`) and `z ↦ z + εu` (moving `(a,b)` by `(Au, Bu)` mod
`Ann`, hence `W` by `2ABu = 0`). ∎(4)

**Remark (general altitude).** Nothing above uses the Koszul shape beyond
commutativity (`ÂB̂ = B̂Â`) and the liftability of the differentials'
matrix entries. The right abstract statement: *if a bounded complex of
free `D`-modules is the mod-`ε̂²` reduction of a complex of free
`F₂[ℤ/4]`-modules, its transfer-LES connecting maps compose to zero* (this
is `d₁² = 0` of the `ε̂`-adic filtration spectral sequence, unwound). The
toy complex `D →^{·ε} D →^{·ε} D` (plan note §1) has `δ₁δ₂ ≠ 0` and
therefore admits no such lift — free-ness + self-duality alone can never
prove the theorem; liftability is the content, and the Koszul complex of
lifted polynomials is liftable *by construction*.

**Remark (the elementary strata).** Two special cases have one-line
proofs not needing the lift, and they explain the sweep taxonomy:
(i) `g = 0` (`ε ∈ (A,B)`): `E = 0` by Koszul annihilation (A12 Lemma 2).
(ii) both `A, B ∈ εR̃`, say `A = εA'`, `B = εB'`: then `a := A'z`,
`b := B'z` are valid choices and `W = 2εA'B'z = 0` directly. Note (ii)
does *not* mean the `δ`'s vanish there — on chain blocks with `g > 0`
both are typically nonzero and the composite dies by exactly this
cancellation (see §3). The genuinely non-elementary stratum is `g > 0`
with at least one of `A, B ∉ εR̃`; the sweeps' "live" counter tracks its
two-sided part (`both ∉ εR̃`).

## 3. The chain-ring case, closed forms (W2)

Let `T = S[Z_{2^r}] ≅ S[t]/(t^N)`, `N = 2^r ≥ 2`, `S = F_{2^d}` — the
block type of every cover whose undoubled coordinate has odd order (A12
§1 CRT: blocks are `S[P]` with `P` the 2-part of `G̃`; odd undoubled
coordinate ⟹ `P` cyclic). The order-2 element is `s = w^{N/2}`, so
`ε = (1+w)^{N/2} = t^{N/2}` exactly. By unit invariance and the swap, a
pair is WLOG `(t^α, t^β)` with `0 ≤ α ≤ β ≤ N` (`t^N := 0`). Closed
forms (S-dimensions; multiply by `d` for F₂-dimensions):

| quantity | value |
|---|---|
| `dim (A,B)` | `N − α` |
| `k̃` | `2α` |
| `k` | `2·min(α, N/2)` |
| `g` | `(α − N/2)⁺` |
| `E` | `2(α − N/2)⁺` |

**Lemma (chain case of Theorem A13, independent of §2).** `E = 2g` on
every chain block. *Proof.* If `α ≤ N/2` then `ε = t^{N/2} ∈ (t^α) =
(A,B)`, so `g = 0` and `E = 0` by Koszul annihilation. If `α > N/2` then
*both* `A, B ∈ (t^{N/2}) = εT`, and the elementary cancellation (§2
remark (ii)) gives `δ₁δ₂ = 0`, hence `E = 2g` by the defect identity. ∎

The `E`-row of the table doubles as an independent check: a direct
syzygy computation (`ker ∂₁` spanned by `(t^{β−α}h, h)`, `(t^{N−α}h, 0)`,
`(0, t^{N−β}h)`; `dim ε·ker∂₁ + im ∂₂ = α` by a triangular-basis count)
yields `E = 2α − N` for `α > N/2`, matching `2g`. Machine-verified: the
orbit-exhaustive sweep covers `F₂[Z₁₆]`, `F₄[Z₈]`, `F₈[Z₄]` completely,
and the `live ≡ 0` phenomenon on chain blocks is exactly this lemma
(`g > 0 ⟹ both ∈ εT`).

**Corollary (odd undoubled coordinate).** For every free-ℤ₂ BB cover
`Z_{2ℓ}×Z_m → Z_ℓ×Z_m` with `m` odd, Theorem A13 parts 1–4 hold by the
chain lemma alone (all blocks chain; CRT splits `E`, `g` blockwise).
This is the corollary OQ2 asked for, and it is independent of the §2
lift argument — belt and braces.

**Screen formula.** Per block, `k̃ = 2d·(min valuation of the pair)`;
summing over blocks predicts the cover `k̃` from block valuations of the
*cover* pair without building the full cover — an A9-screen ingredient.
